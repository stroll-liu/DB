Cursor

sort (spec) { Cursor }

Cursor.js，第 238 行
对文档进行排序。 注意： 除非查询谓词引用要排序的字段之一或使用 Cursor#hint， 否则索引不会用于排序。 根据 MongoDB 的功能，这是为了不排除不包含索引字段的文档。
姓名 类型 描述
spec object 排序规范。
返回：
类型 描述
Cursor
例子 
// No indexes will be used for sorting.
col.find().sort({ x: 1 });
// If x is indexed, it will be used for sorting.
col.find({ x: { $gt: 4 } }).sort({ x: 1 });
// If x is indexed, it will be used for sorting.
col.find().sort({ x: 1 }).hint('x');
